# tic-tac-toe

ブラウザ上で遊べる **マルバツゲーム（Tic-Tac-Toe）** のWebアプリです。  
技術スタックは **PHP / JavaScript / HTML / CSS**。まずはシンプルに「ローカル2人対戦」を実装し、段階的に機能追加していきます。

---

## 目標（MVP）

- 3x3 の盤面でマルバツゲームができる
- プレイヤーは `○` と `×` を交互に置く
- 勝敗判定（勝ち / 負け / 引き分け）
- リセットして再戦できる
- UI は軽量でわかりやすい（PC/スマホ対応）

---

## 技術スタック

- **PHP**: ルーティング（必要なら）/ 将来的な対戦履歴・セッション管理
- **JavaScript**: 盤面操作、ターン管理、勝敗判定
- **HTML/CSS**: 画面レイアウト、デザイン

## ローカル起動方法

1) PHPビルトインサーバーで起動

```bash
cd tic-tac-toe
php -S localhost:8000 -t public
```
ブラウザでアクセス：
	•	http://localhost:8000

---

## 仕様（暫定）
- 盤面: 3x3
- 初期手番: ○
- クリックで空きマスに置ける（既に埋まっているマスは無効）
- 終局後（勝ち/引き分け）は盤面をロック
- リセット ボタンで初期化

---

## 実装メモ（JS）
- state
  - board: string[9]（"" | "o" | "x"）
  - currentPlayer: "o" | "x"
  - winner: "o" | "x" | "draw" | null
- win pattern（0-8）
  - [0,1,2] [3,4,5] [6,7,8]
  - [0,3,6] [1,4,7] [2,5,8]
  - [0,4,8] [2,4,6]
